yilia主题及其他功能的安装

我选择了yilia作为自己博客的主题，yilia看起来简洁，而且使用起来十分方便，只需要进行简单的配置便可以使用。以下是yilia主题具体的配置使用步骤：

  1. 安装yilia： $ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
  2. 配置blog文件夹下的_config.yml：theme:yilia
  3. 根据文档配置yilia下的_config.yml





修改头像、QQ、微信

打开你的项目根目录中blog-theme-yilia-source，在其中可添加一个assets文件夹，里面存放图片资源。然后打开blog-theme-yilia下的_config.yml文件，在其中配置：

# SubNav
subnav:
    github: "https://github.com/dujiajianbing"
    qq: "/assets/qq.jpg"
    weixin: "/assets/wechat.jpg"
    mail: "mailto:wangguanjie1001@163.com"

#你的头像url
    avatar: /assets/head.jpg






字数统计和阅读时长

安装 hexo-wordcount

命令：npm i --save hexo-wordcount


配置文件

在theme-yilia-layout-_partial-post目录下下创建word.ejs文件（ejs文件是什么还不清楚）。
